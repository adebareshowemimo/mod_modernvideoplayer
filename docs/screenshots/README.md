# Screenshots

Screenshots used in the Moodle Plugins Directory listing and the main README.

## Files (expected)

| File | Description |
|------|-------------|
| `01-course-view.png` | Activity rendered on the course page |
| `02-player-focus-mode.png` | Player in focus (distraction-free) mode |
| `03-edit-form.png` | Activity settings form (poster, captions, focus, transcript) |
| `04-transcript-panel.png` | Transcript / captions panel (optional) |

## Capture guidelines

- Format: PNG (preferred) or JPG
- Width: 1280 px or greater
- Max file size: ~1 MB each
- Keep UI in English for the Plugins Directory submission
- Use a clean Moodle theme (Boost) and a sample course named clearly (e.g. "Demo course")

## How to capture on Windows

1. Start XAMPP and open `http://localhost/premium-moodlle`
2. Log in as admin; create a demo course with a Modern video player activity
3. Press `Win+Shift+S` (Snipping Tool) to capture each view
4. Save PNGs into this folder with the names above

## Usage

- Uploaded manually in the plugin submission form at
  https://moodle.org/plugins/register.php (Screenshots section)
- Optionally referenced from the top-level `README.md`:
  `![Course view](docs/screenshots/01-course-view.png)`
